# Práctica Vigenère - UD1 Python

## Descripción

Esta práctica consiste en la implementación completa del cifrado y descifrado con el algoritmo de Vigenère en Python. Permite cifrar y descifrar textos tanto desde consola como desde archivos, aplicando rigurosamente el método clásico de suma y resta módulo 26 sobre letras mayúsculas.

---

## Contenidos

- Funciones para cifrar y descifrar textos con Vigenère.
- Funciones auxiliares para lectura y escritura segura de archivos.
- Control adecuado de errores y validación de entradas.
- Menú interactivo para uso desde consola con opciones para ficheros.
- Documentación completa acorde al estándar PEP 257 y comentarios explicativos.
- Buenas prácticas de programación y organización en proyecto Python.

---

### Requisitos

- Python 3.10 o superior
- (Opcional) pytest para ejecutar los tests

### Ejecutar programa

Desde la consola, en la carpeta del proyecto:

python src/vigenere.py
